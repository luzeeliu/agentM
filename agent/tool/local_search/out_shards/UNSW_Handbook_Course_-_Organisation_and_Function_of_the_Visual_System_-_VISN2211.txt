Description
Objectives
: An understanding of processing in the visual system.
Brief curriculum
: Spatial vision: visual acuity, contrast sensitivity, aliasing, alignment thresholds; Colour vision: retinal and cortical processing, colour constancy, colour identification; Binocularity: models of depth perception, correspondence, Panum's area, horopter, stereoacuity, monocular depth perception, summation, stereopsis tests; Motion perception: models of motion perception, adaptation, directional selectivity, first and second order motion; Cognition: shape recognition, face perception, visual illusions, visual hallucinations, visual attention, visual search and pop-out.
