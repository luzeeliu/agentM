This course introduces students to data visualisation in the areas of statistical analysis and data science. Students will learn how to make their own effective and impactful visualisations using software tools including R. There will be a strong focus on developing the skill of data storytelling: combining data, its visualisation, and a narrative to create a powerful story to drive change. Key topics include visualisation for exploratory and statistical data analysis and interactive visualisations such as dashboards.
